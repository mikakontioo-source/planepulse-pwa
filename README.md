# PlanePulse PWA v1.6

Static Vercel PWA. No npm/build required.

Upload only these files/folders to GitHub root:
- api/
- public/
- index.html
- icon.svg
- manifest.webmanifest
- sw.js
- vercel.json
- README.md

Do not upload package.json, package-lock.json, node_modules, src, dist or server.js.
